# CarootGenCore

Core Minecraft Plugin for the CarootGen (Minekeep) Minecraft Server.

#### 

### **Features:**

**Commands:**

* /lobby						- Sends the executor to the world "lobby".
* /tutorial \[id]					- Shows Basic tutorials for the Server's gamemodes.
* /rootcorereloadconfig				- Reloads the plugin's config.
* /autocomp <admin> <list/add/remove>		- Allows Admins to add 'Auto-compress' recipes for the 'Gens' gamemode, to automatically change an item's type once a certain amount is reached.
* /leaderboard set <kills/blocks/playtime> 	- Allows Admins to place custom leaderboards using the 'DecentHolograms' plugin API.



**Other:**

* Configurable Chat filter via the plugins config.yml
* Auto-Pickup functionality for the Gens gamemode
* Custom  Join / First Join / Leave  Messages
